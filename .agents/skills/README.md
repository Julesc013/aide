# AIDE Repo-Local Skills

A skill in this repository is a narrow reusable instruction set for a recurring work type. Skills are local routing aids, not project law.

## Skill Boundaries

- Skills are narrower than `AGENTS.md`.
- Skills should describe when they trigger, when they do not trigger, and what boundaries they enforce.
- Skills should avoid vague descriptions and should not promise capabilities that do not exist yet.

## Operating Rule

Prefer small, reusable, instruction-first skills that map cleanly onto established repository areas such as inventory, host families, packaging, and evaluation work.
