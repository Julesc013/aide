# Microsoft Visual Studio For Mac

This directory is the archival host-family scaffold for Microsoft Visual Studio for Mac. Child directories represent the MonoDevelop-derived native lane and an AIDE companion fallback lane.

P11 makes the family split explicit:

- `monodevelop-addin`: structural blocked archival-native proof
- `companion`: runnable `cli-bridge` fallback proof

This keeps the retired native lane visible while still giving the family one honest executable first-wave entry point.
