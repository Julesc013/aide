# Microsoft Host Families

This subtree holds Microsoft host-family scaffolds and their compatibility-technology lanes. Product directories stay version-neutral, while exact coverage and lifecycle facts live in `inventory/`, `matrices/`, and `research/microsoft/`.

P11 introduces the first Microsoft boot-slice artifacts. Runnable proofs remain intentionally thin and delegate behavior to the shared-core CLI bridge, while non-runnable native lanes now carry explicit blocked structural records instead of silent placeholders.
