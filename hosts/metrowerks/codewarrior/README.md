# Metrowerks CodeWarrior

This directory is the committed host-family scaffold for Metrowerks CodeWarrior. Child directories represent the documented native `ide-sdk` lane and the AIDE `companion` fallback lane.

P13 turns both lanes into explicit boot-slice proofs:

- `ide-sdk` reaches a runnable archival-native `L1` report-first proof through `cli-bridge` mode and keeps native in-host SDK loading visible through structural metadata
- `companion` reaches a runnable `L1` fallback proof through `cli-bridge` mode for archival or unresolved workflows outside the native lane

Exact version coverage and platform reach are tracked in inventory and matrices.
