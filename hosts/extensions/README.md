# AIDE Extension Hosts

## Purpose

IDE/editor extensions are one class of AIDE Host. Extension hosts are thin OEM/editor shells over AIDE Core.

## Non-Goals

Extension hosts must not own:

- model routing
- repo law
- workflow semantics
- persistent memory
- provider logic

## Boundary

Q02 creates lightweight placeholders only. Existing bootstrap-era host proof lanes remain where they are. Future extension implementation requires separate reviewed queue work.
