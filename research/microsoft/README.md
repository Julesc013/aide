# Microsoft Research

This directory records verified Microsoft IDE ecosystem knowledge that affects AIDE architecture for Microsoft Visual Studio on Windows and Visual Studio for Mac.

Files in this area:

- `sources.md`: source log for official Microsoft material used in this prompt.
- `visual-studio-windows.md`: Windows Visual Studio lineage, lifecycle, and extension breakpoints.
- `visual-studio-mac.md`: Visual Studio for Mac lineage, retirement posture, and extensibility notes.
- `extension-technology-map.md`: mapping between Microsoft extension technologies and AIDE adapter lanes.
- `platform-reach.md`: high-level Microsoft host-platform and target-reach notes used by the platform matrix.
- `open-questions.md`: unresolved or deferred Microsoft ecosystem questions.

These research files are narrative and source-backed. Machine-readable consequences belong in `inventory/` and `matrices/`.
