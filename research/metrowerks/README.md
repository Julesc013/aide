# Metrowerks Research

This directory records verified Metrowerks and CodeWarrior ecosystem facts that matter to AIDE's committed `metrowerks.codewarrior` host family.

Files in this directory cover CodeWarrior lineage, IDE SDK and automation surfaces, extension-lane mapping, platform reach, and explicit unresolved items. Research notes are narrative and source-backed. Machine-readable commitments live separately in `inventory/` and `matrices/`.

This area is factual and architectural. It does not imply that AIDE has implemented a CodeWarrior adapter.
