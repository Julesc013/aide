# Apple Research

This directory records verified Apple-ecosystem knowledge that affects AIDE's Xcode planning.

Files in this area:

- `sources.md`: Apple source log used for this prompt.
- `xcode-history.md`: Xcode lineage and major-family table.
- `xcodekit-surface.md`: public XcodeKit and source-editor-extension surface.
- `extension-technology-map.md`: mapping from Apple facts to AIDE lanes.
- `platform-reach.md`: host and target platform reach relevant to AIDE.
- `current-xcode-notes.md`: current-Xcode architectural posture notes.
- `open-questions.md`: unresolved and deferred Apple items.

These research documents are narrative and source-backed. The normalized outputs of that research belong in `inventory/` and `matrices/`.

This folder is factual and architectural. It is not an implementation area and it does not imply that any Apple lane is already built.
