# Q37 Prompt Summary

Implement Repo Intelligence Index v0.

Q37 builds deterministic, local, no-call indexes for the AIDE repository:
file inventory, file classification, ownership map, dependency map, test map,
documentation link map, generated map, orphan-candidate map, and compact
Markdown summary.

The phase is index-only. It must not move, delete, refactor, migrate, absorb,
install, upgrade, repair, publish, call providers/models, use network calls, or
mutate target repositories. It prepares Q38 File Quality Ledger v0 by making
repo state cheaper and safer to inspect.
