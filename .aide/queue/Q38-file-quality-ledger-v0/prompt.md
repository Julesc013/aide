# Q38 Prompt Summary

Implement File Quality Ledger v0.

Q38 consumes Q37 repo intelligence outputs and produces deterministic,
repo-local, advisory quality evidence for file ownership, kind/status,
generated and evidence boundaries, documentation, tests, validators, stale doc
links, orphan candidates, module-size warnings, mixed-purpose hints, and
reuse/modularity candidates.

The phase is measure-and-report only. It must not move, delete, refactor,
migrate, absorb, install, upgrade, repair, publish, call providers/models, use
network calls, mutate branches, mutate target repositories, or auto-fix files
based on ledger findings. It prepares Q39 Refactor Control Plane v0.
