# Q41 Existing Tool Absorption v0 Prompt Summary

Implement a deterministic, local, no-execution Existing Tool Absorption layer
for AIDE.

Q41 must discover candidate repo tools, classify capabilities and risks, assign
review-only fates, create future wrapper/adaptation plans, add schemas and
policies, expose AIDE Lite `tools` commands, generate advisory outputs, update
golden tasks/tests/docs, synchronize the export pack, and write evidence.

Q41 must not delete, rename, migrate, wrap-execute, or replace tools. Unknown
tools must not be executed. Target repositories must not be mutated. Generated
tool inventories and wrap plans are evidence only and are not target truth.
