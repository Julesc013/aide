# AIDE Bridges

## Purpose

AIDE Bridges are downstream or project-specific overlays that connect AIDE Core semantics to bounded local proof profiles, generated artifacts, companion surfaces, or constrained integration contexts.

Bridges are not provider adapters, tool adapters, protocol adapters, or host implementations. Those may be inputs or consumers, but they are not Bridges by themselves.

## First Bridge

Dominium Bridge is the first intended bridge. Q02 creates skeleton space only; Q07 owns the Dominium Bridge baseline.
