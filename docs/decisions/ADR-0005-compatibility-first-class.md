# ADR-0005: Compatibility Is First-Class

## Status

accepted

## Context

AIDE spans modern and historical development environments. Support, capability, version, platform, migration, and blocker claims need durable handling.

## Decision

Make Compatibility a first-class internal AIDE Core domain.

## Consequences

- Inventory, matrices, research, environment, lab, and eval evidence remain important inputs.
- Candidate hosts and future IDE Hosts cannot be promoted without compatibility evidence.
- Migration work must be planned and reviewed rather than hidden inside source moves.

## Supersedes / Superseded By

Supersedes: none. Superseded by: none.
