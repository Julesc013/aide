# Core Harness

## Purpose

Harness is the future executable repo machinery for deterministic checks and evidence capture around AIDE Core.

## Machinery

- import
- compile
- validate
- doctor
- bakeoff
- migrate
- drift check

## Boundary

Q02 creates the Harness skeleton only. Harness v0 is Q04. Existing shared tests, fixtures, eval records, and queue helpers remain in their bootstrap-era locations until a later reviewed migration exists.
