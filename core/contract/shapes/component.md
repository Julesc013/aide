# Component Declaration Shape

## Purpose

Component declarations identify ownership and review boundaries for the AIDE self-hosting repo.

## Required Fields

- `id`
- `purpose`
- `owned_paths`
- `status`
- `non_goals`
- `verification_posture`

## Optional Fields

- `allowed_references`
- `notes`

## Boundary

Component ownership is conceptual. It does not authorize moving source files or editing forbidden paths.
